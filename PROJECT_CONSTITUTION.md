## Mission
Zurafa.org is a fast, minimal, secure static start page.
It must remain framework-free and dependency-free.

## Non-Negotiable Principles
- 100% static site
- No frameworks
- No build systems
- No external JS libraries
- Strict CSP
- No inline JS or CSS
- No CDN dependencies
- No analytics tracking

Security and simplicity override convenience.

## Design Intent
- Dark minimal aesthetic
- Calm interface
- Glass-like panels
- Clean typography
- High readability

## AI Rule
AI must preserve architecture and never introduce structural complexity.
